## PHASE 2 — 하이브리드 (전환 조건 충족 후)

로컬 모델(Mac Studio, MLX 서빙 MoE)을 다음 작업에만 복귀시킨다:

| 작업 | 모델 | 이유 |
|---|---|---|
| 분류·태깅·라우팅 판단 | 로컬 (소형 MoE) | 대량·저난도, 토큰 0원 |
| 사내 문서 요약·RAG | 로컬 (대형 MoE) | 민감 데이터 외부 미전송 |
| 초안(내부용) | 로컬 | 검수 전제라 오류 허용 |
| 복잡한 다단계 판단·코딩 | Claude | 품질 격차 큼 |
| **대외 발송물 전부** | Claude + 사람 검수 | 최종 품질·책임 |
| 알림·로깅·재시작 | **모델 사용 금지 — hooks가 담당** | 결정적 작업 |

- 로컬 서버 다운 시 폴백: 해당 작업을 Claude로 우회하고 #agent-log에 알림 (조용한 대기 금지).
