# 도메인 정의 템플릿 (🚧 게이트 통과용, 채워서 커밋)

> 목적: 이 문서가 채워지면 D5의 **"파트 정의 테이블"**이 되고, 그 위에서 도메인 종속 파이프라인
> (리드→블로그→견적 / SNS / 학습 / 영상)을 파트별 하드코딩 없이 찍어낸다.
> 작성 원칙: **측정 가능하게.** "브랜드 키운다" ❌ → "월 리드 30건, 블로그 주 2편" ⭕.
> 모르는 칸은 `?`로 두고 넘어가도 된다(빈칸이 곧 다음 질문). 파트는 3개까지 권장(넘치면 우선순위).

---

## PART A · (예시 — PE Studio, 채워진 견본이니 참고용)

| 항목 | 내용 |
|---|---|
| **ID / 이름** | `biz-a` / Paper Engineering Studio (papercraft.kr) |
| **한 줄 정체** | 자기구조설계 특허 기반 페이퍼 아트 제작·납품 (B2B 기념품·전시) |
| **이번 분기 목표(측정가능)** | 신규 리드 월 20건 · 견적 발송 월 8건 · 블로그 주 2편 |
| **타깃 고객** | 백화점·기업 마케팅팀, 전시기획사, 브랜드 굿즈 담당 |
| **담당 자동화** (해당 체크) | ☑리드발굴 ☑블로그 ☑견적서 ☐SNS ☐영상 |
| **리드 수집 소스·키워드** | 네이버/구글: "기념품 제작", "전시 조형물", "브랜드 굿즈 제작"; 공고 사이트 |
| **발행 채널** | 블로그=**WordPress?(확인 필요)** / 네이버는 반자동 |
| **견적 입력** | 품목·단가 테이블(트로피/조형물/포장), PDF 템플릿 있음? `?` |
| **쌓을 데이터** | 리드DB(회사·담당·문의내용·예산·상태) · 발송 견적 이력 |
| **대외 산출물(프런티어 고정)** | 블로그 본문, 견적서 문안, 고객 회신 |
| **승인 게이트** | 견적 발송=필수 · 블로그 발행=필수 |
| **성공지표(KPI)** | 리드→견적 전환율, 견적→수주율 |
| **현재 상태 / 우선순위** | 운영 중(papercraft-studio 코드 존재) / P1 |

## 사업 외 도메인 (도메인 독립 파이프라인 입력 — 이건 게이트 안 기다림)

### 📈 주식 자동매매
| 항목 | 내용 |
|---|---|
| 대상 시장 | ☐국내 ☐미국 |
| 전략 스타일 | (예: 스윙/모멘텀/배당, "?"도 가능) |
| 리스크 한도 | 일 손실 한도 = / 종목당 비중 = / 총 노출 = |
| 계좌 | 한국투자증권 모의→실전 / 초기 투입금 = |
| 킬스위치 조건 | (예: 일손실 -X% 도달 시 전량 정지) |

### 📚 관심 주제 학습 (수집→퀴즈)
| 항목 | 내용 |
|---|---|
| 학습 주제 3~5개 | (예: AI 에이전트, 페이퍼엔지니어링, 마케팅…) |
| 소스 | RSS/뉴스레터/유튜브 채널 목록 |
| 퀴즈 빈도 | (예: 매일 아침 5문항) |

## 작성 후
이 파일을 채워서 커밋하면(또는 채팅에 붙여주면) → 나는 이걸 **파트 정의 테이블(JSON/YAML)**로 변환하고,
D5 공용 골격 + D10~14 종속 파이프라인 설계를 확정한다. **빈칸(`?`)이 많아도 시작 가능** — 채워진 파트부터 만든다.
